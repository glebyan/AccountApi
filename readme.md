The simple service that provide REST API fro basic operation with bank accounts.

#### Feature list

* Async non-blocking read access to account value 
* Thread-safe money transfer between accounts
* Very simple transaction management (restore the accounts state if something goes wrong)
* Roll-back successful money transfer
* Account creation service
* Deposit and withdraw operation for serve ATM
